class cakery.algorithms.DivideAndChoose(users, cake)¶

This is an implementation of the divide and choose algorithm that works as follows:

  1. A cutting user is randomly chosen
  2. The other user is labled as the chooser
  3. The cutter divides the cake into two pieces
  4. The chooser picks the piece they like the best
  5. The cutter gets the remaining piece

class cakery.algorithms.LoneChooser(users, cake)¶

This is an implementation of the lone chooser algorithm that works as follows:

  1. Perform divide and choose with two players
  2. Another player is added to the division
  3. Each current player divides their current share
  4. This should be 1/n (where n is the player count)
  5. The new player chooses the best slice from each player
  6. Repeat at 2 for each new player

Initializes a new instance of the algorithm

class cakery.algorithms.BalancedAlternatingChoice(users, cake, strategy=None)¶

This algorithm takes turns between each user choosing their next favorite item from the resource. It runs as follows:

  1. Flatten the resource into a collection of choices
  2. At each round, have each user choose their best piece
  3. If the pieces are not the same, assign each user that piece
  4. Otherwise, put those pieces in the contested pile
  5. Repeat at 2 until the original pile is exhausted
  6. Now run the simple alternation choice on the contested pile

Initializes a new instance of the algorithm

Parameters:
  • users – The users to operate with
  • cake – The cake to divide
  • strategy – The alternation strategy to use
