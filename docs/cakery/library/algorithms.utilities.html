API Documentation¶

Algorithm Utilities¶

These are a collection of simple helper methods that make the code easier to read (basically a simple DSL).

class cakery.algorithms.utilities.AlternationStrategy[source]¶

A collection of predefined alternation strategies that can be used to supply a user ordering for choosing resources.
